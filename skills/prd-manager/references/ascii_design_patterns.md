# ASCII 原型设计参考

本文档提供了常用的 ASCII 原型设计模式和示例。

## 1. Web 页面布局

### 1.1 基础页面结构

```
┌─────────────────────────────────────────────────────────────┐
│  Logo                          [搜索框]      登录 | 注册     │
├─────────────────────────────────────────────────────────────┤
│  首页  |  产品  |  服务  |  关于我们  |  联系我们           │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│                        主内容区域                             │
│                                                               │
│                                                               │
│                                                               │
├─────────────────────────────────────────────────────────────┤
│  © 2024 公司名称  |  隐私政策  |  服务条款                   │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 侧边栏布局

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo]                                    [用户] [设置]     │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                   │

```

### 1.3 仪表板布局

```
┌─────────────────────────────────────────────────────────────┐
│  仪表板                                    [用户] [通知]     │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│  │  指标1   │  │  指标2   │  │  指标3   │  │  指标4   │   │
│  │  1,234   │  │  5,678   │  │  9,012   │  │  3,456   │   │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘   │

│  │                    数据表格                           │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```
