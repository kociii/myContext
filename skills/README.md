# Claude Skills

> 可复用的 Claude 能力扩展

## Skills 列表

| Skill | 说明 | 路径 |
|-------|------|------|
| PRD 生成器 | 自动生成产品需求文档 | [prd-generator](./prd-generator/) |
| Claude API | Claude API 使用示例 | [claude-api](./claude-api/) |

## 使用方法

1. 将需要的 skill 目录复制到项目 `.claude/skills/` 目录
2. 按照各 skill 的 README 使用

## Skill 开发规范

- 每个 skill 独立目录
- 必须包含 `README.md` 说明
- 可包含示例代码和模板文件
