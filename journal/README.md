# 每日日志

> 对话记录和日常笔记

## 目录结构

```
journal/
├── README.md          # 本文件
├── _index.md          # 年度导航
└── 2026/              # 按年份
    ├── _index.md      # 年度索引
    └── 03/            # 按月份
        ├── _index.md  # 月度索引
        └── 11.md      # 每日日志
```

## 日志格式

```markdown
# 2026-03-11

## 对话主题
1. 主题一
2. 主题二

## 关键产出
- 产出项 1
- 产出项 2

## 收集的资料
- [文章标题](../knowledge/tech/ai/2026-03-11-xxx.md)

## 后续行动
- [ ] 待办事项 1
- [ ] 待办事项 2
```

## 使用说明

1. 每次对话结束时，AI 会自动总结并创建/更新当日日志
2. 日志按 `YYYY/MM/DD.md` 结构存放
3. 月度索引自动汇总当月日志
4. 年度索引汇总全年概览
